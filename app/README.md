# Capacitor

##

```
npm install @capacitor/core
npm install @capacitor/cli --save-dev
```

## Install Capacitor.
```
ng add @capacitor/angular
```

## Build the Web App.
```
ng build --prod
```

## Install the native platforms you want to target.
```
npm install @capacitor/ios @capacitor/android
npx cap add android
npx cap add ios
```
